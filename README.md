# 2023 届 CUMTB 科协巡线车大赛车辆源码

如标题所示，这是 2023 届科协巡线车源码。

车辆使用一块 Raspberry Pi 4B 作为主控，用 640*480@60fps 的摄像头完成赛道识别。

驱动使用大家喜闻乐见的 L298N，用 3S 动力电池供电，主控供电则使用降压板降至 5V 供电。

比赛需要的技术手册在 [document.md](document.md) 中。
